POST request format

    URL - '/create_user'

    INPUT
        image : binary file
        age : int
        gender : int
        handedness : int

    OUTPUT
        image_no: int


GET request format

    URL - '/retrieve_result/<int:image_no>'

    INPUT
        image_no: int (as a parameter)

    OUTPUT
        result: Boolean (True if PD is detected)


status codes used:
    201 - successfully created (POST)
    200 - successful diagnosis (GET)
    400 - missing input (GET/POST)
    204 - not found content (GET)
